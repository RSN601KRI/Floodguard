# FloodGuard — AI-Powered Flood Decision Intelligence System

FloodGuard is a modern SaaS platform designed to transform fragmented environmental data into real-time, actionable decisions during flood crises. Instead of just visualising data, FloodGuard helps authorities **predict risks, plan evacuations, and allocate resources efficiently**.

## Overview

FloodGuard simulates a real-world operational intelligence system inspired by Palantir Foundry + AIP concepts. It integrates multiple data sources and uses AI-driven logic to provide **decision-ready insights** for disaster management teams.

> From **data → intelligence → action**

## Key Features

### AI Decision Engine
- Predicts flood risk based on environmental signals  
- Generates actionable recommendations:
  - Evacuation planning  
  - Resource allocation  
  - Risk alerts  

### 🗺️ Interactive Command Centre
- Real-time map with zone-based risk visualisation  
- Colour-coded alerts:
  - 🟢 Safe  
  - 🟡 Moderate  
  - 🔴 High Risk  
- Clickable zones with detailed insights  

### Ontology-Based Data Model
Structured system connecting:
- Weather events  
- River levels  
- Population data  
- Geographic zones  

Relationships:
- Weather → River → Zone → Population  

### Real-Time Simulation Engine
- Simulates:
  - Heavy rainfall  
  - Rising river levels  
- Dynamically updates:
  - Risk scores  
  - Alerts  
  - Decisions  

### Smart Alerts System
- Generates alerts for:
  - Officials  
  - Field workers  
  - Citizens  
- Simulates SMS / WhatsApp notifications  

### System Control Panel
- Start / Pause monitoring  
- Scenario simulation toggle  
- Adjustable thresholds:
  - Rainfall sensitivity  
  - Risk levels

## Workflow

1. User selects region (State → District → Zone)  
2. Starts monitoring system  
3. Data ingestion begins (simulated real-time)  
4. Risk scores are calculated  
5. AI generates recommendations  
6. User approves actions  
7. Alerts are triggered  

## Demo Flow

1. Normal conditions  
2. Trigger rainfall increase  
3. System detects rising risk  
4. AI recommends evacuation  
5. Alerts sent  
6. Map updates dynamically
